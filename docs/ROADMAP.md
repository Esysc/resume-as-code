# Roadmap

Vision and planned features for CV as Code.

## Current Status (v0.1)

✅ **Stable & Ready**

- Multi-language support (EN/FR/IT)
- Python backend generators (HTML/JSON/PDF)
- React frontend with language selector
- Dark mode toggle
- GitHub Pages deployment
- GitHub Actions CI/CD

## Q1 2026: Foundation

- [ ] Improved HTML template
- [ ] Better PDF styling
- [ ] Pre-commit hooks for validation
- [ ] Community contributions guidelines

## Q2 2026: Features

- [ ] **Resume Parser** - Local CLI to convert PDF → YAML
- [ ] **Multiple Templates** - Choose between layouts
- [ ] **Skills Proficiency** - Add levels (Beginner/Intermediate/Expert)
- [ ] **Certifications with Expiry** - Track certificate validity

## Q3 2026: Polish

- [ ] **Theme Customization UI** - Color and font picker
- [ ] **Experience Timeline** - Interactive visualization
- [ ] **SEO Optimization** - Structured data, meta tags
- [ ] **Git Version Comparison** - View CV changes over time

## Q4 2026 & Beyond

- [ ] **Resume Parser with AI** - Claude API for intelligent extraction
- [ ] **Analytics Dashboard** - Page views, downloads, engagement
- [ ] **Export Formats** - LinkedIn, ATS-friendly plain text
- [ ] **Community Templates** - User-contributed designs
- [ ] **Certifications API** - Link to Credly, verification badges
- [ ] **Testimonials** - Endorsements section
- [ ] **Portfolio Links** - GitHub, portfolio projects
- [ ] **Publications** - Research papers, blog posts
- [ ] **Speaking Engagements** - Conferences and talks
